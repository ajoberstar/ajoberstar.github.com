Gradle Plugins

I've made a few plugins for the Gradle build tool. My active ones are:

  • Gradle-Git - collection of plugins and tasks to interact with Git
  • Gradle-Imports - organizes Java/Groovy import statements (for those who don't like IDEs)

The following ones are deprecated, as they are now in Gradle core.

  • Gradle-Jacoco - plugin to enable Jacoco coverage analysis
  • Gradle-Plugins - Findbugs, JDepend, and PMD plugins

Libraries

  • Grgit - general purpose Git library implemented in Groovy (wrapped around JGit)

Applications

Here's a couple apps I created mainly for my own "productivity". Not that spending weeks or months writing an app necessarily makes you more productive...

  • Todo.txt-HTML5 - todo list web app that uses the todo.txt format
  • Mulima - a music library manager for people who want to control their metadata
